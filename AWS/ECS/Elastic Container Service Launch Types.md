
[[Elastic Container Service Networking]]

- AWS Fargate 
	- You do not manage servers, you manage only the container.
	- Each task hat its own isolation boundary, making it highly secure for sensitive data.
	- Scaling is faster as you don't wait for EC2 instances to boot up.
- EC2 Launch Type
	- You manage a cluster of EC2 instances.
	- Higher complexity but allows for specialized hardware or complex networking requirements.
